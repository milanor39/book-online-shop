# bookonlineshop

使用vue cli、vue-router、vuex製作的網路日文書店，以下連結為上架的作品。

本作品僅為練習作品，不具有任何買賣功能。

[白波書店](https://fabulous-naiad-faae32.netlify.app/#/)

### 會員功能
- 註冊
- 登入
- 修改會員資料
- 查詢訂單
- 查詢願望清單
  * 移除
  * 加入購物車

npm install
